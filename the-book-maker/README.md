### Why it exists

This script was created for a very specific need: reading my own notes ‚Äî and some work documents ‚Äî in a distraction-free environment. Kindle is calm while plain text is not. ***the-book-maker*** sits somewhere in between.

---

### How it works

* Written in R
* Uses `officer`, `dplyr`, and `stringr`
* Applies simple length-based rules to structure the text
* Adds document metadata and a Table of Contents
* Produces a `.docx` file ready to be transferred to Kindle
